#  FitLife

Sistema de gerenciamento de treinos e acompanhamento fitness para usuários de academia.

##  Objetivo
Auxiliar usuários a organizar seus treinos, acompanhar evolução e manter disciplina na rotina fitness.

##  Funcionalidades
- Cadastro e login de usuários
- Criação e edição de treinos
- Controle de exercícios (séries e repetições)
- Acompanhamento da evolucao e histórico
- Dashboard de desempenho
- Notificações, metas e ranking
